# Raef Laffi Portfolio

Premium React portfolio for Raef Laffi, built with Vite, TypeScript, Tailwind CSS, Framer Motion, GSAP, and local project imagery.

## Scripts

```bash
npm run dev
npm run build
npm run lint
```

The page is organized into section components in `src/sections`, shared UI pieces in `src/components`, and portfolio data in `src/data`.
